# PRD - 天大吃什么 Agent (MVP)

## 背景
天津大学北洋园校区学生在校内或校周边就餐时，常常面临信息分散、选择困难、时间紧张的问题。

## 目标
- 输入自然语言需求，快速给出适合北洋园场景的吃饭推荐
- 结构化抽取条件：预算、位置、场景、口味、时间
- 允许同学补充新店和用餐反馈，持续完善校园美食地图

## 目标用户
- 天津大学北洋园校区学生
- 少量覆盖卫津路校区与周边

## 典型输入
- “预算 30，北洋园这边，晚上想吃辣的，和室友一起。”
- “北洋园中午，一个人，清淡点，20 以内。”
- “现在在北洋园，夜宵有什么性价比高的推荐？”

## 交付形态
- REST API
- Web 页面
- 腾讯混元 Lite 结构化推荐
